About

PCB Defect Classifier uses deep learning to detect and classify the defects of a printed circuit board. Based on two methods reference and none-referenced, we train a deep learning model on a Deep-PCB dataset which has 1500 640x640 pcb test-template images in it. This page provides a little bit more information about how the model works and all the results we've gained.

How does PCB Defect Classifier detect defects?

We use 2 methods, one is a 'old school' method of which detect stage and classify stage separated, and has a litter lower on performance but higher at accuracy in some contexts, another is a cutting-edge method which build deep learning model that make detect and classify into only one model.
